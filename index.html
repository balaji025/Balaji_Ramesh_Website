Minimal-Context Stress Tests of Figurative Reasoning in LLMs

NLP · Evaluation

Built a curated benchmark to stress-test how large language models interpret figurative language under minimal context. Developed a data pipeline to extract high-ambiguity idioms from MAGPIE, implemented probabilistic and free-generation evaluation methods, and compared multiple LLMs against human performance. The project analyzes prompting strategies, in-context learning, and extraction techniques to reveal systematic reasoning failures and performance gaps.

IPL Analytics

Full-stack · Data

IPL Analytics is a full-stack web application that presents interactive statistics and insights from the Indian Premier League (IPL), covering all seasons up to 2023. The project features a Flask-based backend that processes and serves IPL data through RESTful APIs, and a Node.js-based frontend that visualizes team-wise and player-wise performance in a clean, user-friendly interface. Designed for cricket enthusiasts and data-driven analysis, the platform demonstrates end-to-end web development, API integration, and sports analytics using real-world datasets.

Video-based cricket-shot classification using modified LRCN approach

CV · Deployment

Developed a deep learning model for cricket shot classification from video using a modified LRCN architecture that combines CNN and LSTM layers to capture spatial and temporal features. The model achieved up to 95% accuracy and was presented at the IEEE SILCON 2024 conference held at NIT Agartala.
